# mtg-api
Metamug Code Execution Dependency

```xml
<dependency>
    <groupId>com.metamug</groupId>
    <artifactId>mtg-api</artifactId>
    <version>1.5</version>
</dependency>
```

# Install lastest dependency

```sh
git clone https://github.com/metamug/mtg-api.git
mvn clean install
```
